Churn Prediction with BigQuery & Vertex AI
==================

This [project](https://github.com/SalmaKazemiRashed/Churn-Prediction-BigQuery-Vertex-AI) demonstrates an end-to-end ML system that enables churn Prediction/reduction and retention-driven business models.
The synthetic data generation, store and feature engineering tasks are conducted in BigQuery using SQL and the model training and batch prediction performed using vertex AI Automl.
